# Пример упаковки складских остатков / ТМЦ

## 1. Назначение документа
Этот документ — демонстрационный пример упаковки складских остатков и ТМЦ перед выводом на рынок. Цель — перевести разрозненные остатки из хаотичного состояния в структурированный B2B deal-file, где покупателю понятны состав партии, состояние, документы, логистика и правила переговоров. Это не завершённый кейс продажи и не отчёт о закрытой сделке.

## 2. Что получает собственник на выходе
- карточку товарной партии в едином формате;
- очищенный реестр позиций / SKU;
- карту целевых покупателей;
- карту рисков покупателя с приоритетами;
- структурированный документальный контур;
- фото/видео-пакет по группам и ключевым позициям;
- переговорную позицию без преждевременных уступок;
- маршрут реализации по шагам;
- план действий на первые 7 дней.

## 3. Входные данные от собственника

| Блок данных | Что запросить | Зачем нужно |
|---|---|---|
| Категория ТМЦ | Типы материалов/товаров по функциональным группам | Понять, кому и в каком сценарии реально интересна партия |
| Перечень позиций | Список номенклатуры по каждой группе | Сформировать базовый контур реестра и убрать «серые зоны» |
| Количество | Объём по каждой позиции | Рассчитать ёмкость спроса и структуру лотов |
| Единицы измерения | Шт., кг, м, л, паллеты и т.д. | Исключить ошибки сопоставления в переговорах и логистике |
| Состояние | Новое/б/у/после хранения/с дефектами | Управлять ожиданиями покупателя до осмотра |
| Срок хранения | Дата производства, остаточный срок, условия хранения | Отсечь нерелевантные запросы и снизить риск споров |
| Упаковка | Заводская, переупаковка, без упаковки | Оценить риск повреждения и стоимость отгрузки |
| Маркировка | Этикетки, коды, читаемость, соответствие номенклатуре | Подтвердить идентификацию позиции на складе |
| Сертификаты / паспорта | Наличие и статус подтверждающих документов | Снизить риски по комплаенсу и допуску в эксплуатацию |
| Документы происхождения | Накладные, УПД, инвойсы, происхождение партии | Проверить юридическую и коммерческую прозрачность |
| Партия целиком или частями | Допустимый формат реализации | Настроить маршрут продаж и защититься от хаотичного cherry-picking |
| Минимальная партия | Нижний порог отгрузки по объёму | Сократить нерентабельные мелкие сделки |
| Склад хранения | Локация, режим, ограничения доступа | Рассчитать сценарий осмотра и вывоза |
| Условия самовывоза | Окна отгрузки, требования к транспорту | Убрать операционные сбои при отгрузке |
| Погрузка | Кто организует, техника, график, стоимость | Прозрачно распределить ответственность и затраты |
| Цена | Базовая ценовая рамка | Подготовить аргументированную переговорную логику |
| Срочность освобождения склада | Реальный дедлайн и операционные ограничения | Выбрать темп вывода и корректный формат сделки |

## 4. Почему складские остатки / ТМЦ могут зависнуть
Партии ТМЦ обычно «зависают», когда:
- нет единого реестра и понятной структуры SKU;
- состояние описано общими словами;
- маркировка и фактические позиции расходятся;
- документы частично есть, но не собраны в систему;
- непонятно, продаётся партия целиком или частями;
- логистика и погрузка не просчитаны;
- цена дана числом без обоснования;
- собственник рано уходит в торг, теряя позицию.

## 5. Карточка товарной партии

| Параметр | Содержание |
|---|---|
| Категория ТМЦ | Указывается по группам (демонстрационный пример) |
| Перечень позиций | Список ключевых позиций с группировкой |
| Количество | Фактический объём по группам и позициям |
| SKU | Коды/артикулы в едином формате реестра |
| Единицы измерения | Приведены к унифицированному виду |
| Состояние | Зафиксировано по каждой группе и критичным позициям |
| Срок хранения | Указаны даты и остаточные сроки, где применимо |
| Упаковка | Тип упаковки и её фактическое состояние |
| Маркировка | Статус читаемости и соответствия реестру |
| Документы | Перечень доступных документов и статус сбора |
| Сертификаты | Наличие, срок действия, статус проверки |
| Формат продажи | Партия целиком / продажа частями по правилам |
| Минимальная партия | Нижний порог сделки по объёму |
| Склад хранения | Локация, режим доступа, окна отгрузки |
| Погрузка | Сценарий погрузки и распределение ответственности |
| Самовывоз | Условия допуска и требования к транспорту |
| Цена | Базовая рамка с логикой расчёта |
| Срочность освобождения склада | Операционный горизонт, влияющий на темп сделки |
| Главная задача | Реализовать партию управляемо, без потери маржи и контроля |

### Сильные стороны
- Структурированный реестр позиций и единые правила идентификации.
- Прозрачная логика состояния, упаковки и маркировки.
- Понятные условия формата продажи и логистики.

### Ограничения
- Часть документов может быть в стадии сбора на старте.
- По отдельным SKU требуется дополнительная сверка на складе.
- Финальные условия зависят от объёма, формата сделки и маршрута отгрузки.

## 6. Карта целевых покупателей

| Сегмент | Зачем ему партия | Что будет проверять | Какой аргумент нужен |
|---|---|---|---|
| Профильное производство | Закрыть потребность в материалах/комплектующих | Совместимость, состояние, документы, стабильность объёма | Предсказуемый вход в производство без лишних рисков |
| Сервисные компании | Быстро обеспечить запас для текущих заказов | Наличие ходовых позиций, срок хранения, отгрузка | Скорость получения и понятная номенклатура |
| Торговые компании | Сформировать товарный запас под перепродажу | Ликвидность SKU, упаковка, маркировка, цена входа | Маржинальность и прогнозируемая оборачиваемость |
| Дилеры / перекупщики | Купить с дисконтом и перераспределить по клиентам | Глубина дисконта, состав лота, возможность частичных выборок | Управляемая сделка с понятной структурой лотов |
| Строительные компании | Закрыть объектные потребности в сжатые сроки | Объём, доставка, соответствие спецификациям | Снижение срывов по срокам снабжения |
| Компании с похожим оборудованием / технологией | Использовать ТМЦ без длительной адаптации | Техническая совместимость, маркировка, документация | Быстрый ввод в действующий технологический контур |
| Региональные покупатели | Сформировать запас с учётом дефицита в регионе | Логистика, упаковка, целостность партии при перевозке | Расчёт полной экономики «до склада покупателя» |
| Покупатели на переработку / утилизацию | Извлечь ценность из неликвидной части | Состав, состояние, объём, условия вывоза | Прозрачные условия выкупа и отгрузки по графику |

## 8. Чек-лист документов
- правоустанавливающие и закрывающие документы по партии;
- товаросопроводительные документы;
- внутренний реестр SKU с привязкой к фактическому хранению;
- сертификаты/паспорта/декларации (если применимо);
- документы происхождения партии;
- регламент доступа, осмотра и отгрузки;
- шаблон фиксации условий частичной продажи.

## 9. Фото- и видео-чеклист
- общий вид зон хранения по категориям;
- фото ключевых групп ТМЦ крупным и общим планом;
- маркировка и SKU на упаковке/паллетах;
- состояние упаковки и потенциальные дефекты;
- видео прохода по складу с привязкой к реестру;
- видео процесса выборки/подготовки к отгрузке (если возможно);
- погрузочная зона, техника и маршрут выезда.

## 12. Коммерческое позиционирование
Слабая формулировка:

> «Продадим складские остатки, объём большой, готовы обсуждать.»

Сильная buyer-facing формулировка:

Партия ТМЦ представлена как структурированный сделочный файл: состав и объёмы по категориям, единый реестр SKU, зафиксированное состояние, статус документов и правила отгрузки. Покупатель заранее понимает формат покупки (целиком/частями), минимальный объём, логику цены и последовательность следующего шага после квалификации и осмотра.

## 14. Переговорная позиция собственника

### Что нельзя раскрывать слишком рано
- Нижнюю границу цены до квалификации покупателя и понимания формата покупки.
- Внутреннюю срочность освобождения склада в формулировках, ослабляющих позицию.
- Готовность отдать лучшие позиции отдельно без общей стратегии реализации партии.
- Полный реестр без квалификации покупателя, если номенклатура коммерчески чувствительна.
- Дополнительные уступки до понимания объёма покупки, логистики и сроков.

### Что можно раскрывать на первом этапе
- Категорию ТМЦ и общую структуру партии.
- Общий объём и доступные форматы продажи: целиком / частями.
- Минимальную партию и базовые условия отгрузки.
- Состояние по категориям на уровне первичной квалификации.
- Наличие документов и общий статус сертификатов.
- Склад хранения, условия осмотра и базовую логику цены.

### Ответ на вопрос “можно купить только часть?”
«Частичная покупка возможна в рамках заранее утверждённой структуры лотов и минимальной партии. Мы не выводим из партии только самые ликвидные позиции вне маршрута реализации, чтобы не разрушить экономику остатка. Если ваш объём и номенклатура соответствуют правилам лота, подготовим конкретное предложение.»

### Ответ на вопрос “почему такая цена?”
«Цена сформирована не как абстрактная цифра, а по факторам: фактическое состояние позиций, общий объём партии, статус документов и сертификатов, качество упаковки/маркировки, срочность освобождения склада, логистика отгрузки и выбранный формат покупки — целиком или частями. После уточнения вашего объёма и направления отгрузки можем предметно обсудить структуру цены.»

### Ответ на вопрос “документы и сертификаты есть?”
«Часть документов и сертификатов уже в наличии и включена в реестр, часть — в процессе дозбора с указанием статуса и сроков. На этапе квалификации предоставляем перечень и подтверждение по доступным позициям, а расширенный пакет показываем после подтверждения интереса и параметров сделки.»

### Ответ на вопрос “кто оплачивает погрузку и доставку?”
«Возможны несколько структур сделки: самовывоз силами покупателя; погрузка силами продавца по согласованному регламенту; доставка отдельной строкой; включение логистики в цену после ясности по объёму и точке назначения. Выбираем вариант, который прозрачен по срокам и экономике для обеих сторон.»

### Ответ на вопрос “можно приехать и посмотреть партию?”
«Да, осмотр возможен по предварительной записи. Мы согласуем окно, правила доступа на склад и ответственного сотрудника. На площадке показываем согласованные группы ТМЦ, маркировку, упаковку и условия отгрузки. Такой регламент нужен, чтобы исключить хаотичные визиты и не нарушать операционную работу склада.»

## 19. Итог
Партия складских остатков/ТМЦ становится управляемым объектом переговоров, когда собственник заранее готовит структурированный состав, состояние, документы, риск-карту, формат отгрузки и правила диалога с покупателем. Этот шаблон нужен для профессиональной подготовки к продаже без завышенных обещаний, без фиктивных результатов и без утверждений о закрытой сделке.
