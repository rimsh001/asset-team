# A&A Asset Team — static website

Официальный статический сайт A&A Asset Team для домена **aateam.ru**.

## Основная информация
- **Тип проекта:** static website (HTML/CSS/JS, без обязательной сборки).
- **Главная страница:** `index.html` (в корне репозитория).
- **Основные стили:** `style.css`, `visual.css`, `seo.css`.
- **Основной клиентский скрипт:** `script.js`.

## Структура (верхний уровень)
- `index.html` — главная страница.
- `*.html` — SEO/контентные страницы и служебные страницы (`privacy`, `404`, `thanks` и т.д.).
- `style.css`, `visual.css`, `seo.css` — стили сайта.
- `script.js` — интерактивность на клиенте.
- `robots.txt`, `sitemap.xml` — SEO-файлы для индексации.
- `_redirects` — правила редиректов (для Netlify-совместимого деплоя).
- `functions/` — serverless-обработчики API (например, форма `/api/lead`, Telegram/MAX маршруты).

## Локальный запуск
Так как сайт статический, достаточно открыть `index.html` в браузере.

Для корректной проверки ссылок и форм рекомендуется запуск через простой локальный сервер, например:

```bash
python -m http.server 8080
```

После запуска откройте:
- `http://localhost:8080/`

## Деплой
Сайт можно стабильно размещать на GitHub Pages / Cloudflare Pages / Netlify.

### Базовые настройки (если без сборки)
- **Build command:** `none`
- **Publish directory:** `/`

### Примечания
- `index.html` должен оставаться в корне репозитория.
- Для корректной работы формы `action="/api/lead"` нужен провайдер serverless-функций (например, Netlify Functions / Cloudflare Pages Functions с соответствующей конфигурацией).
- Если деплой только как «чистая статика» без функций, API-маршруты `/api/*` необходимо отдельно проксировать/реализовать.

## SEO
- `robots.txt` разрешает индексацию и указывает на `sitemap.xml`.
- `sitemap.xml` содержит публичные HTML-страницы сайта.

## Проверка после деплоя
1. Открыть главную страницу и проверить загрузку стилей/шрифтов.
2. Проверить переходы по меню и якорям.
3. Проверить контентные страницы из `sitemap.xml`.
4. Проверить отправку формы (`/api/lead`) на целевой платформе.
5. Проверить `https://aateam.ru/robots.txt` и `https://aateam.ru/sitemap.xml`.
